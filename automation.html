Automation & Mechatronics Projects

Explore my PLC, mechatronics, and industrial automation projects

PLC Conveyor Belt Control

Designed and implemented a PLC-controlled conveyor belt system for sorting objects based on color and size. Developed ladder logic using timers, counters, and sensors to ensure precise automation. The system efficiently detects incoming objects, sorts them into designated paths, and maintains smooth workflow, demonstrating practical industrial automation concepts and real-time control.

Automated Pick-and-Place

Developed a 4-axis robotic arm controlled by PLC for automated pick-and-place operations. Integrated position and proximity sensors to detect objects and avoid collisions. Designed ladder logic to coordinate arm movements, timing, and gripping mechanisms, providing a robust solution for high-precision material handling in industrial automation scenarios.

Smart Packaging Line

Implemented a fully automated packaging line using PLCs and multiple sensors for quality and process control. Designed an HMI interface for real-time monitoring, start/stop control, and workflow adjustments. The system handles product detection, alignment, and packaging, demonstrating integration of automation, safety protocols, and industrial process efficiency.

Mechatronic Door Control

Designed an automated door control system using PLC and servo motors for industrial applications. Integrated safety sensors and timed operations to prevent accidents and ensure smooth functioning. The project involved ladder logic programming to control opening/closing sequences, emergency stop mechanisms, and remote monitoring, combining mechanical and electrical automation concepts.

Temperature Control System

Implemented a PID-controlled heating system using PLC to maintain precise temperature levels. Integrated temperature sensors for real-time feedback and an HMI for monitoring setpoints and current readings. Designed ladder logic for dynamic adjustment, safety cut-offs, and alarm triggers, demonstrating practical applications of control theory and automation in industrial heating processes.

Line Following Robot

Designed and built an Arduino-based autonomous line following robot. It uses IR sensors to detect and follow a predefined path with high accuracy. Implemented proportional speed control to navigate curves smoothly and avoid deviations. The project demonstrates fundamental robotics concepts including sensor integration, motor control, and real-time decision-making.

Couple Tank Water Equilibrium

Implemented an Arduino-controlled couple tank system to maintain water level equilibrium between two tanks. Equipped with water level sensors and a PID controller to regulate the inflow and outflow accurately. The system ensures stable levels even under varying input disturbances, demonstrating real-time control and process automation skills. Ideal for illustrating fundamentals of control engineering and fluid dynamics.

RC Car using Bluetooth

Developed a Bluetooth-controlled RC car using Arduino and a motor driver shield. The car can be controlled remotely through a smartphone app for forward, backward, left, and right motion. Integrated obstacle detection using ultrasonic sensors to prevent collisions. This project showcases embedded system programming, wireless communication, and real-time sensor feedback implementation.

Smart Suitcase

Developed an Arduino-based smart suitcase that autonomously follows the passenger using sensors and motor control. Integrated ultrasonic and IR sensors to detect and maintain a safe distance from the user. The suitcase navigates obstacles, turns, and stops automatically, showcasing real-time tracking, autonomous navigation, and embedded system programming. Ideal for demonstrating practical applications of robotics in daily life.
